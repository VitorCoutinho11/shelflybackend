Sobre o Projeto

O **Shelfly** é uma solução móvel (Android/iOS) desenvolvida para leitores. Nosso objetivo é **simplificar a rotina de leitura**, oferecendo ferramentas para:

  • Acompanhamento de progresso de leitura.
  • Histórico e gerenciamento de títulos lidos e pendentes.
  • Ajuda a manter a consistência e o foco diário.
Desenvolvido com React Native e Expo.

  3. Pode aparecer um aviso sobre arquivos APK. Toque em OK ou **Manter** para prosseguir com o download.
